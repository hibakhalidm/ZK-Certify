## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/your-repo/trustseal
cd trustseal
```

2. Install dependencies:
```bash
npm install
```

3. Set up environment variables:
```bash
cp .env.example .env
# Edit .env with your credentials
```
